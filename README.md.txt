# Pneumonia Detection using VGG19

This project detects Pneumonia from chest X-ray images using deep learning.

## Model
- VGG19 (Transfer Learning)
- Fine-tuned on Chest X-ray dataset

## Dataset
- Chest X-ray dataset (Normal / Pneumonia)

## Accuracy
- Test Accuracy: ~90%

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn

## How to Run
1. Install requirements
2. Run the Jupyter Notebook
3. Load model and test on X-ray image
